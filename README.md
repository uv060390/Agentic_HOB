# Agentic_Marketing

A platform to hire specialised AI Agents for each category in D2C Marketing.

## Status

> **Work in Progress** — This project is currently under active development and is expected to be completed by **2026-03-19**.

## Overview

Agentic_Marketing enables D2C brands to leverage specialised AI Agents for each marketing category, streamlining campaign management, performance tracking, and audience targeting.
